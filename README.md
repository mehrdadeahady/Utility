# Utility — Unified UI for gRPC‑Driven Microservices

> **Utility** is a modular, extensible UI designed to interact with multiple backend microservices over **gRPC**.  
> It provides a unified interface for service orchestration, debugging, and operational workflows.

---

## ✨ Features

- **Unified UI** — One interface for multiple microservices  
- **gRPC Integration** — Strongly typed, high‑performance communication  
- **Modular Architecture** — Add or remove service modules easily  
- **Service Discovery** — Auto‑detect or configure endpoints  
- **Real‑time Streaming** — Supports gRPC streaming for live updates  
- **Developer Tools** — Logs, request inspector, schema viewer
